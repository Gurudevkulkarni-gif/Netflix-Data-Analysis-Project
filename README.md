# 🎬 Netflix Data Analysis Project

## 📌 Overview
This project analyzes Netflix dataset using Python to discover insights about movies and TV shows available on Netflix.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Trend Analysis

---

## 🚀 Features
- Analyze Netflix content by year
- Compare Movies vs TV Shows
- Genre analysis
- Country-wise content analysis
- Ratings analysis
- Visualization using charts and graphs

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Files in Project

- `netflix_data.ipynb` → Main analysis notebook
- `mymoviedb.csv` → Dataset file
- `README.md` → Project documentation

---

## 📊 Visualizations Included
The notebook contains multiple visualizations such as:
- Bar Charts
- Count Plots
- Heatmaps
- Pie Charts
- Trend Graphs


---

## 📈 Key Insights
- Movies are more frequent than TV Shows
- Certain genres dominate Netflix content
- Content growth increased rapidly after 2015
- USA contributes highest content on Netflix
